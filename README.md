# The Wealth Blueprint
### *Strategic Analytics for Smarter Investing*

## Overview
**The Wealth Blueprint** is a professional-grade financial simulator designed to provide a realistic view of long-term wealth creation. Developed as part of my **PGDM in Finance & Analytics**, this tool goes beyond standard calculators by incorporating real-world variables that impact an investor's net worth.

## Expert Features
- **Dynamic Step-up Engine:** Models the impact of annual investment increases (matching career growth).
- **Taxation Intelligence:** Integrated 2024-25 Indian LTCG tax logic (12.5% tax on gains exceeding â‚¹1.25L).
- **Purchasing Power Analytics:** Real-time inflation adjustment to show the future corpus in today's value.
- **The 'Cost of Delay' Module:** A behavioral finance feature quantifying the wealth lost by procrastinating.
- **Risk-Adjusted Portfolios:** Pre-set asset allocations for Conservative, Balanced, and Aggressive profiles.

## Technical Stack
- **Backend:** Python 3.x
- **Frontend:** Streamlit (Light & Professional Theme)
- **Data Visualization:** Plotly Interactive Charts
- **Library Dependencies:** Pandas, NumPy

## How to Run
1. Access the live dashboard via the link in the 'About' section.
2. Adjust your monthly SIP, tenure, and step-up percentage in the sidebar.
3. Analyze the 'Wealth Gap' between a standard SIP and a Step-up SIP.
